Документация API

Базовый URL: {{ api_base_url }}

Метод Endpoint Описание
GET /api/v1/employees/ Список сотрудников
GET /api/v1/departments/ Список отделов
GET /api/v1/events/ HR-события (дни рождения, отпуска, истечение договоров)
GET /api/v1/documents/ Кадровые документы
GET /api/v1/company/ Данные компании
POST /api/v1/auth/token/ Получить токен (username + password)

Примеры использования

Получить список сотрудников:

curl -H "Authorization: Token {{ token.key }}" \
     {{ api_base_url }}employees/

Получить токен по логину и паролю:

curl -X POST {{ api_base_url }}auth/token/ \
     -d "username=your@email.com" \
     -d "password=your_password"

Python:

import requests

headers = {"Authorization": "Token {{ token.key }}"}
response = requests.get("{{ api_base_url }}employees/", headers=headers)
employees = response.json()
