Политика конфиденциальности

Дата вступления в силу: 16 апреля 2026 г. · Редакция 1.0

Настоящая Политика конфиденциальности описывает, какие персональные данные собирает сервис «Кадровый автопилот», как они используются и защищаются. Использование Сервиса означает согласие с условиями настоящей Политики.

1. Оператор персональных данных

Оператором персональных данных является ИП Малых Евгений Николаевич, ИНН 270317039817, г. Комсомольск-на-Амуре, Хабаровский край, Российская Федерация.

Контакт по вопросам обработки персональных данных: support@kadrovik-auto.ru

2. Какие данные мы собираем

Данные пользователя (аккаунт):

  • Адрес электронной почты — для идентификации, входа в систему и уведомлений.
  • Название компании — для персонализации интерфейса.
  • Дата регистрации и история активности в Сервисе.

Данные сотрудников (вводимые Пользователем):

  • ФИО сотрудников, должности, даты приёма/увольнения.
  • Паспортные данные (если вводятся для формирования документов по ТК РФ).
  • Данные о структурных подразделениях, табели рабочего времени, приказы об отпусках.

Технические данные:

  • IP-адрес, тип браузера и операционной системы — для обеспечения безопасности.
  • Логи действий в системе — для диагностики и улучшения Сервиса.

3. Цели обработки данных

  • Предоставление доступа к Сервису и формирование кадровых документов.
  • Отправка уведомлений, связанных с работой Сервиса (напоминания о событиях, окончание подписки).
  • Улучшение качества Сервиса и исправление ошибок.
  • Исполнение обязательств по договору (Оферте).
  • Соблюдение требований законодательства Российской Федерации.

4. Правовые основания обработки

Обработка персональных данных осуществляется на следующих основаниях:

  • Согласие субъекта персональных данных (при регистрации в Сервисе).
  • Исполнение договора, стороной которого является субъект персональных данных.
  • Требования законодательства Российской Федерации.

5. Хранение и защита данных

Все данные хранятся на серверах, расположенных на территории Российской Федерации, в соответствии с требованиями Федерального закона № 152-ФЗ «О персональных данных».

Для защиты данных применяются:

  • Шифрование соединения (HTTPS/TLS).
  • Хранение паролей в виде хеша (без возможности восстановления в открытом виде).
  • Разграничение доступа: данные компании доступны только авторизованным пользователям этой компании.
  • Регулярное резервное копирование.

6. Передача данных третьим лицам

Мы не продаём и не передаём персональные данные третьим лицам в коммерческих целях. Ограниченный доступ к данным может предоставляться:

  • Хостинг-провайдеру (Timeweb Cloud) — исключительно для обеспечения работы инфраструктуры Сервиса.
  • Платёжному сервису ЮKassa (ООО «НКО ЮМани») — данные, необходимые для обработки платежей.
  • Telegram — при использовании функции уведомлений (только идентификатор чата, без персональных данных сотрудников).
  • Государственным органам — по законным требованиям в соответствии с законодательством РФ.

7. Права пользователей

В соответствии с Федеральным законом № 152-ФЗ каждый Пользователь вправе:

  • Получить информацию о составе и целях обработки своих персональных данных.
  • Потребовать уточнения, блокирования или удаления своих персональных данных.
  • Отозвать согласие на обработку персональных данных.
  • Экспортировать данные своей компании из Сервиса.

Для реализации своих прав обратитесь по адресу: support@kadrovik-auto.ru. Запрос будет обработан в течение 30 дней.

8. Удаление данных

При закрытии аккаунта данные Пользователя и его сотрудников сохраняются в течение 30 дней, после чего удаляются. Пользователь вправе запросить немедленное удаление, направив письмо на email Оператора.

9. Cookies

Сервис использует технические cookies, необходимые для работы сессий аутентификации. Они не содержат персональных данных и удаляются по истечении сессии или при выходе из аккаунта. Маркетинговые и аналитические cookies не применяются.

10. Изменение Политики

Оператор вправе вносить изменения в настоящую Политику. Актуальная версия всегда доступна по адресу kadrovik-auto.ru/privacy.html. Продолжение использования Сервиса после публикации изменений означает согласие с новой редакцией.

11. Контакты

По любым вопросам, связанным с обработкой персональных данных, обращайтесь: support@kadrovik-auto.ru
